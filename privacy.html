Privacy Policy

Last updated: 2025-09-29

Aegis Scan is a privacy‑first API that performs static inspection of uploaded PDF files to identify risky features (e.g., JavaScript, embedded files, encryption). This policy explains what data we process, how we handle it, and your choices.

1. What we process

  • Uploaded files: Files are received solely to perform the requested inspection and are not retained after the response is produced.
  • Derived metadata: We compute flags/metrics (e.g., has_javascript, risk_score) and return them in the API response. We may aggregate anonymous metrics for service quality (e.g., counts of feature flags).
  • Operational logs: Minimal request metadata is logged for abuse prevention and rate‑limiting (timestamp, API key ID, IP hash, request path, response status, latency). No file contents are stored in logs.

2. Retention

Uploaded files are processed in memory/ephemeral storage and discarded immediately after analysis completes. Operational logs are retained for a limited period needed to enforce rate limits, detect abuse, and operate the service.

3. Security

Transport security (HTTPS) is required. Access is authenticated with API keys. Internal access to operational data is restricted. We periodically review dependencies and infrastructure configurations. Report issues via hello@aegisscan.dev or our security.txt.

4. Cookies & tracking

Our static marketing site may use basic analytics (page views, referrers). The API endpoints themselves do not use cookies.

5. Children’s privacy

The service is intended for business and developer use. We do not knowingly collect personal data from children.

6. International use

We may process data in the region where our hosting provider operates. Business plan customers may request region‑specific endpoints.

7. Your choices

  • Do not send personal data unless necessary. Prefer redacted or test files.
  • Use your API key management to rotate/disable keys at any time.
  • Contact us for questions or deletion requests relating to operational logs.
