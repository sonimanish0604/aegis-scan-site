Data Handling

Last updated: 2025-09-29

This page explains how Aegis Scan processes files and related metadata when you call the API, including retention, locations, and security controls.

1. Scope & Purpose

Aegis Scan performs static inspection of PDF files to identify risky features (e.g., JavaScript, embedded files, encryption, external links). We provide a structured JSON result so you can automate allow/deny actions.

2. What We Process

  • Uploaded files: received solely to perform analysis and not retained after the response is produced.
  • Derived metadata: flags and counts returned in the API response (e.g., has_javascript, risk_score, details.urls).
  • Operational logs: minimal request metadata for reliability and abuse prevention (e.g., timestamp, API key ID, IP hash, route, status, latency). No file contents are stored in logs.

3. Retention

  • Files: processed in memory/ephemeral storage and discarded immediately after analysis completes.
  • Operational logs: retained for up to 30 days to enforce rate limits, investigate abuse, and operate the service.
  • Aggregated metrics: may be kept in anonymized/aggregated form for service quality and capacity planning.

4. Locations (Data Residency)

By default, processing occurs in the region where our hosting is deployed. Business plan customers may request region‑specific endpoints.

5. Security Controls

  • Transport security via HTTPS; API authentication via keys.
  • Least‑privilege access to operational data; secrets stored in provider secret stores.
  • Dependency and configuration reviews; rate‑limiting and abuse detection.

6. Subprocessors

  • Hosting: Render (application/API hosting).
  • Caching/rate‑limiting: Managed Redis (environment‑specific).
  • We do not use third‑party storage for uploaded file contents.

7. Incident Reporting

Report security issues to hello@aegisscan.dev and see our security.txt for contact and coordination details.

8. Changes

Material changes will be noted on this page and may be announced on release notes.
